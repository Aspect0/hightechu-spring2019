# Ajax and Loopback

* https://loopback.io/

## How to get started

1. Clone the repository onto your local machine.

2. Change your directory into the git repository.

```bash
cd ajax-loopback
```

3. Change your directory into the loopback folder.

```bash
cd loopback
```

4. Install all dependecies.

```bash
npm install
```

5. Run Server Locally

```bash
node .
```
